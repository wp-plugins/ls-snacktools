== Description ==
Add oEmbed support for Slidesnack.com in WordPress posts, pages and custom post types.
There are no settings. Simply, add the Slidesnack URL in your content editor.
Simply paste the given "Link to this post" from Slidesnack.com  into the content area and it will by transformed to oEmbed into your post, page or custom post type.

Example: Copy http://snack.to/bhcs4ppx and paste it into your post. 


== Installation ==
1. Go to Dashboard >> Plugins >> Add New >> Search
2. Search for "LS Snacktools" click "Install Now," then "Activate Plugin"
3. There are no settings.

OR

1. Upload `ls_snacktools` folder into the `/wp-content/plugins/` directory

2. Activate the plugin through the 'Plugins' menu in WordPress
